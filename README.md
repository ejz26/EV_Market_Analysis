# 🚗 Tesla's Next Move: Using Data Analytics to Identify the Biggest Opportunity in the EV Market

## Overview

This project analyzes the electric vehicle market to identify growth opportunities and recommend Tesla's next vehicle using Python, SQL, and Power BI. Through competitor analysis, pricing trends, and vehicle performance, the project provides data-driven insights to support strategic product decisions.

## Objectives

- Analyze trends in the EV market
- Compare Tesla with major competitors
- Identify underserved market segments
- Determine the features that drive sales and customer satisfaction
- Recommend Tesla's next vehicle based on data

## Tools & Technologies

- Python (Pandas, NumPy, Matplotlib, Scikit-learn)
- SQL (PostgreSQL)
- Power BI
- Git & GitHub
